Web Tasarım Dilleri

HTML

açılımı HypertextMarkup Language( Hiper Metin İşaretleme Dili ) olan web sayfası oluşturmak için kullanılan işaretleme dilidir. Birçok kişi bunu dil olarak kabul etmemektedir. Fakat html olmadan çalışan bir web uygulaması olamaz.

Css

açılımı Cascading Style Sheets( Basamaklı Stil Şablonları) olan ve html yapıları üzerinde renk, boyut, konumlandırma gibi özellikleri ayarlayan bir web teknolojisidir.

Javascsript

geliştiricilerin etkileşimli web sayfaları oluşturmak için kullandığı bir programlama dilidir.
